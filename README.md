# Quality Signal Hub 2.0

Modern dark-themed **Quality Intelligence Cockpit** for artifact runs generated by the quality automation tool.

## Run dashboard (local mode)
1. `npm run dashboard -- --run results/<runId> --port 4173`
2. Open `http://localhost:4173`

Local mode serves:
- UI shell
- `/api/model` normalized discovered data model
- `/artifacts/...` image assets from the selected run directory

## Run dashboard (static mode)
1. `npm run dashboard:build`
2. `npm run dashboard:serve -- --run results/<runId> --port 4173`

Static mode serves prebuilt front-end files from `dist/dashboard` while still reading artifacts from `--run`.

## Dashboard capabilities
- Executive overview across all URLs (scores, status, release gate context)
- Portfolio with search/filter/sort and score charts
- Trends view from `history.json`
- URL drilldown with sections:
  - Executive
  - Performance
  - Accessibility
  - Security
  - Network
  - Stability
  - SEO
  - Visual regression
- Compare view for URL deltas
- CI Health governance panel from `ci-summary.json`

## Documentation
- `STATUS.md`: phase-by-phase implementation checklist and acceptance self-checks
- `DATA_MAPPING.md`: UI-to-artifact mapping and fallbacks
- `CONFIG.md`: scoring weights and thresholds
